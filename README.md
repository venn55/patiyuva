PatiYuva — Hazır, gezilebilir demo sitesi (deploy-ready)

Nasıl kullanılır (lokal):
1. ZIP'i indirin ve tüm dosyaları aynı klasöre çıkarın.
2. Klasördeki index.html dosyasını çift tıklayarak veya tarayıcıda açarak siteyi gezebilirsiniz.
3. Hayvan detay sayfalarına tıklayarak site içinde gezinebilirsiniz.

GitHub Pages deploy (önerilen):
1. Yeni bir GitHub repository oluşturun (ör. patiyuva-demo).
2. Klasördeki tüm dosyaları repoya yükleyin.
3. GitHub repository ayarlarında Pages > Branch: main (veya gh-pages) seçin. Site otomatik yayınlanır.
4. Yayınlanan URL genellikle: https://<kullaniciadi>.github.io/<repoadi>/

Notlar:
- Tüm görseller Unsplash üzerinden hotlink edilmiştir (demo amaçlı). Yayına almadan önce yerel dosyalara taşıyın veya CDN izinlerini kontrol edin.
- Formlar demo amaçlıdır; gerçek e-posta/DB bağlantısı eklemek isterseniz backend örneği ekleyebilirim.
